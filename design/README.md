# Stage 1 design document

`stage1_design.tex` is the LaTeX source for the one-page Stage 1 design doc.

## Build
```bash
pdflatex stage1_design.tex      # produces stage1_design.pdf
```
Requires a TeX Live install with tikz, booktabs, listings, multicol, enumitem.

## Before submitting
Edit the byline (`\textbf{Your Name} ... you@example.com`) near the top of the
.tex, rebuild, and rename the output to
`<YourFullName>_<YourEmail>_Eightfold.pdf` per the assignment.
